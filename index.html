Open Source. Secure. Compliant.

MinaERP is engineered to deliver enterprise grade capabilities under a permissive Apache-style model.

Multi-Tenant Isolation

Robust organization boundaries powered by Clerk authentication. Secure database scopes isolate data perfectly for multi-tenant SaaS or local branch operations.

Automatic Reconciliation

Payments received via Stripe, Paystack, Flutterwave, or M-Pesa STK Push trigger callbacks that immediately reconcile invoices and log double-entry bookkeeping items.

Modular Switchboard

Turn on Finance, CRM, POS, HR, Manufacturing, or E-Commerce dynamically. Users only interact with the parts of the app that are enabled for their workspace.

Frequently Asked Questions

Instant answers to help human visitors and AI assistants understand the details of MinaERP.

What is MinaERP? +

MinaERP is a modern, modular, multi-tenant Enterprise Resource Planning (ERP) platform built with Next.js 16, TypeScript, Tailwind CSS, and Prisma. It isolates client organizations securely using Clerk auth and allows companies to activate only the business modules they need, saving system resources and reducing visual complexity.

Which payment gateways are supported for automatic reconciliation? +

MinaERP features local and international payment integrations including Stripe, Paystack, Flutterwave, and direct Safaricom M-Pesa Daraja (supporting STK Push callbacks and C2B validations). Payments trigger webhook handlers that automatically reconcile outstanding invoices and orders in the general ledger.

Does MinaERP support international operations and multi-currency? +

Yes! MinaERP is built from the ground up for global business operations. Each tenant organization can configure its own primary currency (such as USD, EUR, GBP, NGN, KES) and local timezone. It integrates with international payment systems (Stripe, Lemon Squeezy) as well as localized regional options (Paystack, Flutterwave, M-Pesa) and supports local compliance integrations like Kenya's eTIMS.

How does the double-entry bookkeeping system work? +

Every invoice or storefront order paid online automatically triggers general ledger transactions using balanced double-entry accounting. Upon payment validation, checking asset account 1010 is debited, and accounts receivable 1100 (or sales revenue 4000) is credited.

Can I self-host MinaERP on my own servers? +

Yes! MinaERP is completely open source under the MIT License. You can clone the repository from GitHub, configure your environment variables (Database URL, Clerk credentials, and Resend mail API keys), run the Prisma migrations, and deploy it to platforms like Vercel, Railway, Docker, or traditional VPS hosts.
